# Local Hero Finder

Emergency response network connecting people in need with verified first responders in their area.

## Features

- One-tap SOS emergency alerts
- Real-time location tracking
- Verified responder network
- Live map integration with OpenStreetMap
- Real-time incident updates

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Tech Stack

- React 18
- TypeScript
- Vite
- Supabase
- Leaflet/OpenStreetMap
- Tailwind CSS
